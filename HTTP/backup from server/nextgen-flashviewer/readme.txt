== Description ==

The famous Adobe Flash Plugins (SimpleViewer, TiltViewer, AutoViewer, PostcardViewer) from 
Airtight Interactive for NextGEN Gallery. Go to the [our forum](http://shabushabu-webdesign.com/support-forum/) for support and more information.

Check [NextGEN ImageFlow](http://wordpress.org/extend/plugins/nextgen-imageflow/) for another add-on for your NextGEN Gallery installation.

== Installation ==

1. Upload the files to wp-content/plugins/nextgen-flashviewer

2. Go to the web page http://www.airtightinteractive.com/
   and download the SimpleViewer & Tiltviewer & AutoViewer & PostcardViewer

3. Rename viewer.swf in the PostcardViewer folder to pcviewer.swf

4. Upload the file viewer.swf, TiltViewer.swf, autoviewer.swf and pcviewer.swf to the wp-content/plugins/nggflash-swf folder, which you will have to create

5. Activate the plugin

6. Go to Gallery->FlashViewer and change the options to your liking

7. Go to your post and enter tag [simpleviewer id="GALLERY-ID" width="WIDTH" height="HEIGHT" link="LINK"], [tiltviewer id="GALLERY-ID" width="WIDTH" height="HEIGHT" link="LINK"], [autoviewer id="GALLERY-ID" width="WIDTH" height="HEIGHT" link="LINK"] or [pcviewer id="GALLERY-ID" width="WIDTH" height="HEIGHT" link="LINK"]

8. To use TiltViewer with images from your Flickr account, enter your user ID in the settings and enter a tag like this [tiltviewer id="GALLERY-ID" width="WIDTH" height="HEIGHT" link="LINK" flickrtags="FLICKR,TAGS,SEPERATED,WITH,COMMAS"]

That's it ... Enjoy!
